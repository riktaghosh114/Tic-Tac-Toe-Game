# Tic-Tac-Toe-Project

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.
This project allows two players to play the classic game directly in the browser with a clean and responsive design.

**🚀 Features**

🎯 Two-player gameplay (Player X vs Player O)

🖥️ Responsive UI that works on desktop and mobile

🔄 Restart/Reset button to start a new game

✨ Highlighting of winning combinations

🕹️ Smooth and minimal design for a better user experience

**🛠️ Technologies Used**

HTML5 – for game structure

CSS3 – for styling and responsiveness

JavaScript (ES6) – for game logic and interactivity
